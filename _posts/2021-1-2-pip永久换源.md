### pip 永久换源

由于某些未知原因，天朝的计算机学子难以访问到外国的网站，因此再使用pip 下载一些python库的时候就会报错
这里提供一种比较方便的方式更换源

```
win + R
```
输入%HOMEPATH%
打开后，新建pip文件夹
然后创建pip.txt
将下面的内容写进去
```
[global]
timeout = 6000
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
trusted-host = pypi.tuna.tsinghua.edu.cn
```
改成pip.ini
大功告成！
