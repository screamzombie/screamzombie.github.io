# pip 换源

## 临时使用
```
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple/ name
```

## windows 永久换源
由于某些未知原因，天朝的计算机学子难以访问到外国的网站，因此再使用pip 下载一些python库的时候就会报错
这里提供一种比较方便的方式更换源

```
win + R
```
输入%HOMEPATH%
打开后，新建pip文件夹
然后创建pip.txt
将下面的内容写进去
```
[global]
timeout = 6000
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
trusted-host = pypi.tuna.tsinghua.edu.cn
```
改成pip.ini

## linux 永久换源

修改~/.pip/pip.conf（没有就创建）修改index-url 到tuna：
```
[global]
 index-url = https://pypi.tuna.tsinghua.edu.cn/simple
```

# PyCharm 配置 anaconda 环境
## 配置 anaconda 环境变量

**在系统环境变量Path添加anacond的安装路径和其子文件夹script和library/bin**
类似下面的，不过具体路径要改成本机的
吐槽一下，当年wyd居然还不会配置环境变量
```
G:\Anaconda3;
G:\Anaconda3\Scripts;
G:\Anaconda3\Library\bin;
```
配置好后重启计算机

点击PyUIC之后，会自动生成python文件,打开生成的python文件,还需要在最后面添加下面的代码才能运行
```
import sys
from PyQt5.QtWidgets import QApplication, QMainWindow
if __name__ == '__main__':
    app = QApplication(sys.argv)
    MainWindow = QMainWindow()
    ui = Ui_MainWindow()
    ui.setupUi(MainWindow)
    MainWindow.show()
    sys.exit(app.exec_())
```
