TS 的数据类型

JavaScript 共有 string ,number ,boolean,null,undefined,bigint,symbol,object

其中 object 包含:Array，Function,Date,Error 等

TS 在 JS 的基础上多了六个新类型

any,unknown,never,void,tuple,enum

  1. unknown 会强制开发者进行类型检查
  2. nerver 多用于永远不会返回的函数(递归),或者不返回的函数(抛出异常)
  3. undefined 是 void 可以接受的一种空,void 会 return undefined
