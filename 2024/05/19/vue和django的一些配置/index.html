Django е®үиЈ…

python 3.11 д№ӢеҗҺйңҖиҰҒдҪҝз”ЁиҷҡжӢҹзҺҜеўғе®үиЈ…пјҢеҗҰеҲҷе°ұдјҡжҠҘй”ҷгҖӮе…·дҪ“зҡ„еҒҡжі•жҳҜиҝҷж ·зҡ„

йҰ–е…ҲдҪҝз”ЁиҝҷдёӘе‘Ҫд»ӨеҲӣе»әдёҖдёӘ python иҷҡжӢҹзҺҜеўғ venvжҳҜиҷҡжӢҹзҺҜеўғеҗҚеӯ— еҗҺйқўзҡ„жҳҜи·Ҝеҫ„еҸҜд»ҘиҮӘе·ұжүҫдёҖдёӘзӣ®еҪ•ж”ҫиҝӣеҺ»

python -m venv /path/to/new/virtual/environment

жҺҘзқҖиҝӣеҺ»иҝҷдёӘеҲҡеҲҡеҲӣе»әеҘҪзҡ„зӣ®еҪ•еҺ» source е®ғ
жҜ”еҰӮжҲ‘зҡ„з”өи„‘зӣ®еүҚзҡ„иҷҡжӢҹзҺҜеўғең°еқҖеңЁ

~/.venv/

жү§иЎҢиҝҷдёӘе‘Ҫд»Ө

source ~/.venv/bin/activate

иҝҷж ·иҝҷдёӘз»Ҳз«Ҝзҡ„зҺҜеўғе°ұеҸҳжҲҗиҷҡжӢҹзҺҜеўғ python дәҶ

еҰӮжһңдёҚдҪҝз”ЁиҷҡжӢҹзҺҜеўғ pip е®үиЈ…DjangoдјҡзӣҙжҺҘжҠҘй”ҷпјҢе…¶д»–зҡ„дёҖдәӣеҢ…д№ҹжҳҜеҗҢзҗҶ

pip install django

Django дёҺ mysql йҖҡи®ҜйңҖиҰҒдҪҝз”Ё pymysql иҝҷдёӘеә“пјҲжҜ”иҫғиҖҒдәҶпјүпјҢж–°зҡ„еә“й…ҚзҪ®зҺҜеўғжңүдәӣй—®йўҳжүҖд»ҘжҜ”иҫғжҺЁиҚҗдҪҝз”ЁиҝҷдёӘ

sudo pip install pymysql
