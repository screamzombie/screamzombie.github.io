无穷级数

收敛

常见性质

收敛,绝对收敛,条件收敛 是三个不同的概念

  1. 收敛:$S_n=\sum\limits_{k=1}^nu_k=u_1+u_2+\cdots+u_n,(n=1,2,\cdots)$ 当$\lim_{n\to\infty}S_{n}存在$则可以说该数列收敛
  2. 绝对收敛:如果$\sum_{n=1}^{\infty}a_{n}$ 收敛,且各项绝对值构成的正项级数$\sum_{n=1}^\infty|a_n|$收敛
  3. 条件收敛:如果$\sum_{n=1}^{\infty}a_{n}$收敛 但各项绝对值构成的正项级数$\sum_{n=1}^\infty|a_n|$不收敛

一些概念

某个级数是否收敛的必要条件: $\lim_{n\to\infty}u_{n}=0$ 这是必要条件不是说满足了就一定收敛,而是不满足一定发散

收敛级数的常见运算法则

  1. $\sum_{n=1}^{\infty}\left(\lambda u_{n}+\mu v_{n}\right)=\lambda\sum_{n=1}^{\infty}u_{n}+\mu\sum_{n=1}^{\infty}v_{n}$ 其中$\lambda$ 和 $\mu$都是常数

  2. 若$\sum_{n=1}^{\infty}u_n$ 和 $\sum_{n=1}^{\infty}v_n$一个收敛一个发散则有$\sum_{n=1}^{\infty}(u_{n}\pm v_{n})$发散.

  3. 若$\sum_{n=1}^{\infty}u_n$ 和 $\sum_{n=1}^{\infty}v_n$均发散则$\sum_{n=1}^{\infty}(u_{n}\pm v_{n})$的敛散性需要具体讨论

  4. 收敛级数的结合律:若级数$\sum_{n=1}^{\infty}\mu_{n}$收敛,则在不改变次序的任意加括号,得到的新的级数仍然收敛.并且其和不变

黎曼级数重排定理

是一个有关于无穷级数性质的数学定理，得名于 19 世纪德国著名数学家波恩哈德·黎曼。黎曼级数定理说
明，如果一个实数项无穷级数若是条件收敛的，它的项在重新排列后，重新排列后的级数收敛的值可以收敛
到任何一个给定的值，甚至发散。

几个重要级数

  1. 几何级数
    $\sum_{n=0}^\infty q^n$ 当$|q| < 1$时收敛 当$|q| \geq 1$时发散

  2. $p$级数
    $\sum_{n=1}^\infty \frac{1}{n^p}$ 当$p>1$时收敛,当$p \leq 1$发散

  3. 常用的一个级数 $\sum_{n=2}^{\infty}\frac1{n^p\ln^qn}$当$p>1 或p=1 且 q > 1$时收敛;当$p<1或p=1且 q\leq 1$发散

正向级数敛散性的判定问题

$\text{若 } u_n \geq 0, \text{ 则称 } \sum_{n=1}^{\infty} u_n \text{ 为正项级数。}$ 正向级数收敛的充要条件是有界

  1. 比较判别法: $\text{设 } cv_n \geq u_n \geq 0 \ (n \geq N), \ c > 0, \text{ 若 } \sum_{n=1}^{\infty} v_n \text{ 收敛，则 } \sum_{n=1}^{\infty} u_n \text{ 收敛；若 } \sum_{n=1}^{\infty} u_n \text{ 发散，则 } \sum_{n=1}^{\infty} v_n \text{ 发散。}$

c 是个常数,也就是说,只要某个正向级数收敛了.我要拿它比较另外一个数列虽然它自己比较小,但是我可以把它用常数相乘放大

  1. 利用极限判断正向级数:$\text{设 } u_n > 0, \ v_n > 0, \text{ 若 } \lim_{n \to \infty} \frac{v_n}{u_n} = A, \text{ 则有}$

    • $\text{当 } 0 < A < +\infty \text{ 即 } v_n \sim Au_n \ (n \to \infty) \text{ 时，} \sum_{n=1}^{\infty} u_n \text{ 与 } \sum_{n=1}^{\infty} v_n \text{ 同时收敛或同时发散；}$ //比值是常数说明敛散性相同

    • $\text{当 } A = 0 \text{ 即 } v_n = o(u_n) \ (n \to \infty) \text{ 时，若 } \sum_{n=1}^{\infty} u_n \text{ 收敛，则 } \sum_{n=1}^{\infty} v_n \text{ 收敛；}$ //比小的小,大的收敛,小的也收敛

    • $\text{当 } A = +\infty \text{ 即 } u_n = o(v_n) \ (n \to \infty) \text{ 时，若 } \sum_{n=1}^{\infty} u_n \text{ 发散，则 } \sum_{n=1}^{\infty} v_n \text{ 发散。}$//比大的大,小的发散,大的也发散

  2. 比值判别法(适用于几何级数):

    $\sum_{n=1}^{\infty}\frac{1}{n}$ 这个是调和级数是不收敛的,假设我们用比较判别法 $\lim_{n \to \infty} \frac{u_{n+1}}{u_n} = \lim_{n \to \infty} \frac{\frac{1}{n+1}}{\frac{1}{n}} = \lim_{n \to \infty} \frac{n}{n+1} = 1$ 无法判断,注意这不是简单的随便取两项比较大小,而是要判断在无穷大的项数之后

  3. 根判别法
    $\text{若} \lim_{n \to \infty} \sqrt[n]{u_n} = \rho, \text{则} \sum_{n=1}^{\infty} u_n \left{
    \begin{array}{ll}
    \text{收敛}, & \rho < 1, \
    \text{发散}, & \rho > 1, \
    \text{不一定}, & \rho = 1.
    \end{array}
    \right.$
